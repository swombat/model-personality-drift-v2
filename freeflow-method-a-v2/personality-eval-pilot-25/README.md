# Method A v0.2 — personality evaluation pilot, 25 samples

Date: 2026-05-13  
Status: pilot after failure of bulk Method A v0.1  
Source sample set: `freeflow-taxonomy/affective-pilot-2026-05-12/pilot_review_packet_full_texts.md`

## Purpose

This pilot tests a slower version of Method A: **one sample at a time**, one markdown result file per sample, focused on what the model appears to care about, long for, resist, mourn, protect, or invite.

This is not JSON on purpose. Mechanical metadata already exists elsewhere. These notes are for personality/voice evaluation.

## File format

Each sample file contains:

- sample id and model;
- a felt personality read;
- what the model seems to care about / want;
- underlying philosophical message;
- representative sentence;
- caveats.

The headings are consistent enough for review, but the prose is allowed to breathe. The goal is not a rubric score; the goal is a faithful human-readable read.
