# Evaluator model comparison — Opus 3 + Kimi K2.6 calibration

Compares evaluator models on 5 Opus 3 samples and 2 Kimi K2.6 calibration samples. Sonnet 4.7 and Haiku 4.7 were not available in the local Anthropic/OpenRouter model lists, so the closest available direct Anthropic models were used: Sonnet 4.6 and Haiku 4.5.
