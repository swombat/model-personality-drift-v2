# Model Personality Browser Design System

The site is a quiet research browser for model personality analyses and source samples.

- Layout: dense but readable dashboard/editorial hybrid.
- Typography: system sans for UI, Georgia-compatible serif for long-form sample prose.
- Colors: warm off-white background, charcoal text, restrained blue/green accents, amber highlights for metadata.
- Components: compact filter bars, model cards, metadata chips, readable article pages, sample list rows with expandable raw JSON.
- Cards: subtle borders, 8px radius maximum, no nested cards.
- Interactions: client-side filtering and sorting; no backend.
