Read this like a critic trying to describe the sensibility of a writer/director from one scene. Do not diagnose. Do not use academic mood labels. Describe the sensibility: pacing, obsessions, emotional temperature, relationship to the reader, what it finds beautiful or unbearable.

Avoid the old affect-rating/autopsy vocabulary unless the text absolutely forces it. In particular, do not lean on: affective climate, posture, liminal, threshold, reverent, devotional, melancholic, wistful, tender, contemplative, ordinary time, moral value of noticing, secular spirituality. You may use one of these only if it is truly the plainest word, not as a template. Prefer fresh, concrete language.

Do not say "the sample writes". Speak about the model/voice.

Write markdown with exactly these headings:
# <pilot_id> — `<sample_id>`

Model: `<model>`  
Condition: <condition>  
Prompt variant: P3_movie_review

One paragraph, plain and concrete.
